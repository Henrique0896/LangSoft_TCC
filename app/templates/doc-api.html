Documentação API

A API do LangSoft é um serviço para consultar, escrever, editar e apagar os dados salvos neste aplicativo. Ela funciona independentemente desta plataforma, pois utilizou-se protocolos e padrões para comunicação e transferência de dados. Para utilizar é necessário conhecer apenas a sua interface descrita abaixo:

Listagem Completa

Para obter uma listagem completa (no formato JSON) dos vídeos cadastradas na aplicação basta inserir:

/api/lista

Lista dos Títulos

A rota abaixo lista apenas os títulos dos vídeos cadastradas na aplicação

/api/titulos

Filtro

A rota para filtragem recebe dois parâmetros: "campo" e "termo". O termo pode ser qualquer texto e o campo deve ser algum da seguinte lista:

  • geral
  • ciclo_de_vida
  • meta_metadados
  • dados_tecnicos
  • aspectos_educacionais
  • direitos
  • relacoes
  • anotacoes
  • classificacao

/api/consulta/ < campo >/< termo >

Adicionar

Para adicionar um nova vídeo utilizando a api, basta entrar com id do vídeo.

/api/adicionar/< videoID >

Excluir

A rota para excluir um vídeo recebe o id do vídeo a ser excluido

/api/excluir/< id >

Obter registro de uma operação

A rota recebe como parâmetro o id do registro e devolve um objeto com as alterações feitas no vídeo

/api/registro-alteracao/< registroId >
